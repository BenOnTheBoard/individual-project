About Us

Motivation

Matching problems arise whenever an assignment must be found between agents based on their individual preferences. Large-scale examples include the U.S. National Resident Matching Program, which matches junior doctors to hospitals, so efficient algorithms are necessary. Algmatch seeks to help teach those with no prior knowledge the purpose and function of stable matching algorithms.

PyPI Module

The Algmatch Python package complements this web app. It provides a centralised collection of efficient and well-tested implementations of algorithms for finding stable matchings in fundamental bipartite problems. It appears, to the best of our knowledge at the time of writing, to be the only publicly available module or library that implements the known polynomial-time algorithms for problems which allow agents to express indifference.
